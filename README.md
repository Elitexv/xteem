
# 📚 XteemLibrary

XteemLibrary is a modern digital e-library platform that allows users to explore books, borrow available titles, request books, and return borrowed books seamlessly. It also includes a powerful admin dashboard for managing books, users, and borrowing activities.

---

## 🚀 Features

### 👤 User Features
- 🔍 Browse available books in the library
- 📖 View book details (title, author, category, availability)
- 📥 Request to borrow books
- 📚 Borrow books instantly if available
- 🔄 Return borrowed books
- 📜 View borrowing history
- ❤️ Save favorite books *(optional future feature)*

### 🛠️ Admin Dashboard
- ➕ Add new books
- ✏️ Edit book information
- ❌ Remove books
- 📦 Track borrowed and returned books
- 👥 Manage users
- 📊 View library statistics
- ✅ Approve or reject borrowing requests
- 🔔 Notifications for pending requests
